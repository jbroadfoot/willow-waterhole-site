# Willow Waterhole Greenspace Conservancy — Website

Static site (HTML/CSS/JS). Hosted on Netlify, source on GitHub.

## Structure
- `index.html` — homepage
- `visit.html` — Plan Your Visit (to be added)
- `prairie.html` — Native Coastal Prairie & Conservation
- `css/styles.css` — shared design system (edit once, applies everywhere)
- `js/main.js` — shared behavior (mobile nav, etc.)
- `images/` — optimized image library
- `_redirects` — legacy URL redirects (301)
- `404.html`, `robots.txt`, `sitemap.xml`

## Editing workflow
1. Describe the change to Claude → get updated file
2. Paste into GitHub web editor (or upload)
3. Netlify auto-deploys in ~30 seconds

## Brand
- Green PMS 348 `#008444` · Blue PMS 300 `#0063BE`
- Fonts: Libre Baskerville (display) + Nunito (body)
- Logo: full-color on white only; white logo on near-black only
